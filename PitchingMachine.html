Pitching Machine

Wednesday, April 20, 2022 - 12:00 AM

Background

This is a pitching machine which I made as a placeholder for an actual pitcher for the baseball game I'm working on.

The model is based on the Atec M3 pitching machine

Software used

  • Blender
  • Substance Painter
  • Substance Designer
  • Gimp

Process

I created the model doing hard-surface modeling in Blender, then I used Substance Designer to create the base textures for the materials. I was using product images as a reference which led to a very clean and kind of boring look, like if it was fresh off the manufacturing line.

This was never going to be a very prominent asset, but I wanted to make it a bit more interesting so I imported it into Substance Painter and hand painted all the rust, dust, and general deterioration of the paint and metals.
