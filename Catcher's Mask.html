Catcher's Mask

Monday, November 08, 2021 - 12:00 AM

This mask is part of a series of baseball assets that I'm working on for a future baseball videogame I plan on developing. The mask is based on the Force3 Pro catchers mask which is designed to deaden the impact of high velocity pitches to prevent concussions by using a unique spring based suspension system.

I made this mask completely in blender using splines to create the cage geometry, and the shader editor to create procedural materials which I then baked into 4 textures at 512x512 to keep the whole thing at a web friendly 330KB.

The current model has 4,268 triangles and a single material with baked diffuse, metallic, roughness, and normal, plus one additional material with a single alpha texture for the springs. I made each texture an atlas so that I could get away with using only 4 textures for the entire model. There are still a lot of design decisions to make for the game itself which would affect how I would go about optimizing these assets but I do know that I want to include a replay system with a free-fly camera which means the player will be able to get a close up view of this mask and other assets. For this reason, I opted to fully model the wire cage even though an alpha texture over simplified geometry would have been sufficient since the main view of the game is a fixed camera with a narrow FOV, far away from the asset itself.

Moving forward, I will likely make significant changes to the asset once I decide what platforms I'm going to target. I'll redo the textures, create LODs, include the spring material in the single texture atlas for one less draw call.
